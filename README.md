# Bjørg de Vibe - Kunstgalleri

Dette er den offisielle nettsiden og det digitale galleriet for kunstner Bjørg de Vibe. Nettsiden fungerer som en portefølje som viser frem verk, utstillingshistorikk og biografi.

## Om prosjektet
Nettsiden er bygget som en moderne, minimalistisk galleri-opplevelse med fokus på kunsten. Den har et responsivt design som fungerer like godt på mobil som på desktop.

- **Portefølje:** Oversikt over malerier i olje og akryl.
- **Utstillinger:** Tidslinje over tidligere og kommende utstillinger.
- **Biografi:** Informasjon om kunstnerens bakgrunn og utdanning.

## Teknisk Stack
- **Framework:** [React](https://reactjs.org/) (TypeScript)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Styling:** Vanilla CSS (Custom properties for tema og layout)
- **Hosting:** [Vercel](https://vercel.com)

## Lokal utvikling

### Forutsetninger
- Node.js installert

### Installasjon
1. Klone depotet:
   ```bash
   git clone https://github.com/DIN_BRUKER/VibeArt.git
   ```
2. Installer avhengigheter:
   ```bash
   npm install
   ```
3. Start utviklingsserver:
   ```bash
   npm run dev
   ```

## Distribusjon
Siden er konfigurert for automatisk distribusjon via Vercel ved push til `main`-bransjen på GitHub.
